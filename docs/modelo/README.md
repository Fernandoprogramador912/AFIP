# Modelo Excel — Recupero tasa de estadística

Archivo versionado: **`Modelo_Excel_Recupero_Tasa_Estadistica.xlsx`**

Es la referencia de columnas (incluida la hoja `recupero_V2`) respecto a la cual se alinea el export `tasa-arca export-recupero` y el panel local.

- **Verde / datos AFIP:** lo que el proyecto intenta completar desde liquidaciones y JSON AFIP/MOA.
- **Gris / fórmulas:** cálculos del modelo manual (base, CIF, excedentes, «A solicitar», etc.).

Si actualizás el modelo en Excel, reemplazá este archivo y revisá que `RECUPERO_V2_HEADERS` en código siga alineado.
